# Тесты для проекта Synergy406todo

## Ручные тесты (Manual Test Cases)
1. Добавление задачи → задача появляется в списке
2. Отметка как выполненная → зачёркивается
3. Редактирование → текст меняется после Enter
4. Удаление → задача исчезает
5. Фильтры All / Active / Completed работают корректно
6. Данные сохраняются после перезагрузки страницы
7. Работает на мобильных разрешениях

## Автотесты (Vitest)
Запуск: `npm run test`

Покрытие:
- Pinia store (addTodo, toggleTodo, deleteTodo, editTodo, фильтры)
- TodoItem.vue (рендер, события toggle/delete/edit)
- TodoList.vue (интеграция со стором)
